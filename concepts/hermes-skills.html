Hermes Skill 索引与触发机制

本页面汇总当前 Hermes Agent 环境中的所有 Skill，按分类组织，并给出每个 Skill 的触发场景与常用命令。 总计 Skill 数量：85

使用方式

Hermes Skill 的触发方式主要有两种：

  1. 自动触发：当用户的自然语言匹配某个 Skill 的描述或触发条件时，Hermes 会自动加载并使用该 Skill。
  2. 主动调用：用户可以明确说「加载 xxx skill」或「使用 xxx」来强制指定使用某个 Skill。
