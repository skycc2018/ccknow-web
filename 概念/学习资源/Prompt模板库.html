Prompt 模板库

按岗位分类的 Prompt 标准模板

管理层模板

经营分析

【背景】
我是[公司名]的[职位]，负责[业务领域]。
当前面临[具体决策场景]。

【角色】
请你扮演一位资深[行业]战略顾问，具备20年[领域]经验。

【输入材料】
- 业务数据：[粘贴关键数据]
- 市场环境：[粘贴市场信息]
- 约束条件：[预算/时间/资源限制]

【任务】
请帮我[具体分析需求]。

【输出格式】
1. 核心发现（3-5条）
2. 可选方案（A/B/C）
3. 各方案利弊对比表
4. 风险评估矩阵
5. 推荐方案及理由
6. 下一步行动清单

HR模板

JD生成

背景：需招聘[高级产品经理]
角色：资深招聘专家
输入：
- 岗位核心职责：[列出3-5条]
- 必备技能：[列出关键要求]
- 公司文化：[团队氛围/价值观]
- 薪酬范围：[预算区间]

任务：生成一份吸引目标候选人的JD
要求：
1. 标题吸引力评分（1-10）
2. 职责描述清晰具体
3. 突出公司差异化优势
4. 避免"负责相关工作"等模糊表述
5. 包含候选人成长路径描述

产品模板

PRD框架

背景：需撰写[功能模块]的PRD
角色：资深产品专家（腾讯/阿里P8级别）
输入：
- 功能目标：[解决什么问题]
- 用户故事：[作为...我想要...以便...]
- 业务流程：[简述]
- 涉及系统：[列出]

任务：生成完整PRD框架
输出要求：
1. 版本记录
2. 背景与目标
3. 用户故事地图
4. 功能详述（含交互逻辑）
5. 数据需求
6. 埋点方案
7. 发布计划
8. 风险清单

运营模板

活动方案

背景：需策划[618大促]活动
角色：资深活动运营（阿里/京东背景）
输入：
- 目标：GMV 500万，新客获取1万
- 预算：50万（含补贴/投放/物料）
- 时间：6月1日-6月18日
- 品类：[列出主力品类]
- 用户：25-35岁女性为主

任务：生成完整活动方案
输出要求：
1. 活动主题与创意
2. 玩法设计（主玩法+子玩法）
3. 节奏安排（预热/爆发/返场）
4. 资源分配表
5. 风险预案（3套）
6. 效果预估模型

销售模板

客户画像

背景：准备拜访新客户[XX公司]
角色：行业顾问 + 销售专家
输入：
- 公司公开信息：[官网/财报/新闻]
- 行业趋势：[行业报告要点]
- 已知联系人：[职位/背景]

任务：生成客户画像
输出：
1. 公司战略优先级（Top 3）
2. 可能痛点（5个）
3. 决策链分析（关键人/影响人/使用人）
4. 竞品使用情况
5. 我方切入点
6. 初次拜访策略

研发模板

功能开发

背景：需开发[用户权限管理]模块
角色：资深后端工程师
输入：
- 语言：Python/Java/Go
- 框架：Django/Spring/Gin
- 数据库：MySQL/PostgreSQL
- 需求：
  - RBAC权限模型
  - 支持数据级权限
  - 接口级鉴权
  - 操作日志

任务：生成完整实现
输出：
1. 数据库设计（ER图描述）
2. 核心代码（含注释）
3. 接口设计（RESTful）
4. 单元测试
5. 性能考虑
6. 安全加固点
