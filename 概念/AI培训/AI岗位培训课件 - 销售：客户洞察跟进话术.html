AI岗位培训课件 - 销售：客户洞察/跟进话术

课程目标

让销售人员掌握用AI分析客户需求、优化跟进策略、提升转化效率的方法

一、销售AI应用场景概览

场景 AI能力 预期效果
客户画像分析 自动分析客户背景，生成画像 客户理解深度提升
跟进话术生成 基于客户阶段生成个性化话术 回复效率提升3倍
商机预测 分析历史数据，预测成交概率 资源聚焦高价值客户
异议处理 生成常见异议的标准应答 转化率提升20%
合同/方案生成 自动生成定制化方案/合同 方案产出时间节省80%

二、核心技能：Prompt Engineering

2.1 标准Prompt模板（销售专用）

【背景】
我是[公司名]的[销售经理]，负责[行业/区域]客户。
当前跟进[客户名称/阶段]。

【角色】
请你扮演一位资深销售专家，擅长[行业]大客户销售。

【输入材料】
- 客户信息：[公司规模/行业/痛点]
- 沟通记录：[上次沟通要点]
- 竞品情况：[已知竞品介入情况]
- 我方优势：[差异化能力]

【任务】
请帮我[具体需求]。

【输出格式】
1. 客户洞察
2. 策略建议
3. 具体话术
4. 风险预警
5. 下一步行动

【校验标准】
- 符合商务礼仪
- 不夸大承诺
- 可落地执行

2.2 高频场景模板

场景1：客户画像分析 - 背景：准备拜访新客户[XX公司] - 角色：行业顾问 + 销售专家 - 输入： - 公司公开信息：[官网/财报/新闻] - 行业趋势：[行业报告要点] - 已知联系人：[职位/背景] - 任务：生成客户画像 - 输出： 1. 公司战略优先级（Top 3） 2. 可能痛点（5个） 3. 决策链分析（关键人/影响人/使用人） 4. 竞品使用情况 5. 我方切入点 6. 初次拜访策略

场景2：跟进话术生成 - 背景：客户[XX]处于[方案确认]阶段，已3天未回复 - 角色：资深销售 - 输入： - 客户角色：[采购经理] - 上次沟通：[方案已发，对方表示需内部讨论] - 已知顾虑：[价格/交付周期] - 任务：生成跟进话术 - 输出： 1. 3种跟进策略（温和/直接/价值强化） 2. 每种策略的话术模板 3. 客户可能的回复及应对 4. 升级时机判断 5. 备选方案（如客户沉默超1周）

场景3：方案/合同生成 - 背景：客户[XX]需求已明确，需输出正式方案 - 角色：解决方案专家 - 输入： - 客户需求：[列出5-8条] - 我方能力：[匹配的产品/服务] - 商务条件：[预算范围/交付时间] - 任务：生成方案框架 - 输出： 1. 客户痛点共鸣 2. 解决方案架构 3. 实施路径 4. 投资回报分析 5. 成功案例（类似客户） 6. 下一步推进计划 - 格式：可直接发给客户的PPT大纲

三、实操案例

案例1：大客户首次拜访准备

背景： 销售小张要拜访行业头部客户，但准备时间只有半天。

解决方案：

客户研究Prompt

请基于以下信息，生成客户深度画像：
客户：[XX集团]
行业：[制造业]
规模：[年营收100亿/员工5000]
公开信息：[粘贴]

要求：
1. 业务板块分析
2. 近期战略动向
3. 数字化水平评估
4. 可能需求场景（5个）
5. 我方最佳切入点
6. 初次拜访谈话框架

话术准备Prompt

基于以上画像，请准备：
1. 开场白（建立共鸣）
2. 需求探询问题（10个）
3. 价值陈述（3个场景）
4. 常见抗拒及应对
5. 收尾与下一步
6. 需避免的话题

效果： - 准备时间：2天 → 半天 - 客户专业度感知：一般 → 非常专业 - 初次拜访成功率：30% → 70%

案例2：沉默客户激活

背景： 多个客户进入"沉默期"，不知如何有效激活。

解决方案：

沉默原因分析Prompt

以下是[5个]沉默客户的情况：
[粘贴客户信息+沟通记录]

请分析：
1. 沉默可能原因分类
2. 每个客户的激活难度
3. 最佳激活时机
4. 激活话术策略
5. 放弃标准（何时止损）

激活话术生成Prompt

请为以下客户生成激活话术：
客户类型：[预算冻结/竞品介入/需求变化/内部变动]

输出：
1. 激活邮件/微信（3个版本）
2. 电话脚本
3. 价值点重新包装
4. 优惠/活动借力方案
5. 跟进节奏（频次/内容）

效果： - 激活率：10% → 35% - 平均激活周期：2个月 → 2周 - 资源浪费减少：大量 → 精准

案例3：招投标方案快速产出

背景： 临时接到招标通知，5天内需提交完整方案。

解决方案：

招标分析Prompt

请分析以下招标文件：
[粘贴招标要求]

输出：
1. 招标方真实需求（显性+隐性）
2. 评分标准权重分析
3. 我方优势/劣势
4. 主要竞争对手预判
5. 方案策略建议
6. 需澄清的问题

方案生成Prompt

基于以上分析，请生成投标方案：
约束：5天内完成

输出：
1. 技术方案框架
2. 实施计划
3. 团队配置
4. 成功案例匹配
5. 差异化亮点
6. 风险承诺

格式：可直接排版的标准结构

效果： - 方案产出时间：2周 → 3天 - 方案完整度：70% → 95% - 中标率提升：+15%

五、常见误区与避坑指南

误区1：AI生成的方案直接发给客户 → 必须人工审核，确保数据准确、承诺可兑现。

误区2：用AI完全替代客户沟通 → AI辅助准备，但真诚的人际互动不可替代。

误区3：过度承诺 → AI可能生成过于乐观的方案，需人工把关。

误区4：泄露客户机密 → 客户敏感信息输入AI需脱敏，注意保密。

  3. 案例沉淀：将本次实操经验整理为团队SOP
