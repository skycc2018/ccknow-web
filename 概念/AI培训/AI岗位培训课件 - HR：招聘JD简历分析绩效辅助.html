AI岗位培训课件 - HR：招聘JD/简历分析/绩效辅助

课程目标

让HR掌握用AI提升招聘效率、优化人才评估、辅助绩效管理的方法

一、HR AI应用场景概览

场景 AI能力 预期效果
智能JD生成 基于岗位需求自动生成/优化招聘文案 撰写时间节省70%
简历智能筛选 自动提取关键信息，匹配度评分 筛选效率提升10倍
面试辅助 生成面试问题、评估候选人回答 面试质量标准化
薪酬分析 市场薪酬数据对比，内部公平性分析 薪酬决策有据可依
绩效辅助 绩效数据汇总，改进建议生成 绩效面谈效率提升

二、核心技能：Prompt Engineering

2.1 标准Prompt模板（HR专用）

【背景】
我是[公司名]的HR[职位]，负责[招聘/绩效/薪酬]模块。
当前需要处理[具体场景]。

【角色】
请你扮演一位资深HR专家，具备10年[行业]人力资源管理经验。

【输入材料】
- 岗位信息：[职位/级别/部门]
- 候选人资料：[简历/面试记录]
- 公司要求：[文化/价值观/硬性条件]

【任务】
请帮我[具体需求]。

【输出格式】
1. 核心结论
2. 详细分析
3. 风险提示
4. 建议行动

【校验标准】
- 符合劳动法规
- 避免性别/年龄/地域等歧视性描述
- 数据保密，不泄露敏感信息

2.2 高频场景模板

场景1：招聘JD生成 - 背景：需招聘[高级产品经理] - 角色：资深招聘专家 - 输入： - 岗位核心职责：[列出3-5条] - 必备技能：[列出关键要求] - 公司文化：[团队氛围/价值观] - 薪酬范围：[预算区间] - 任务：生成一份吸引目标候选人的JD - 要求： 1. 标题吸引力评分（1-10） 2. 职责描述清晰具体 3. 突出公司差异化优势 4. 避免"负责相关工作"等模糊表述 5. 包含候选人成长路径描述

场景2：简历分析与匹配 - 背景：收到[50份]简历，需快速筛选 - 角色：资深HR + 业务面试官 - 输入： - 岗位JD：[粘贴] - 候选人简历：[粘贴] - 任务：评估匹配度 - 输出： 1. 匹配度评分（1-100） 2. 核心优势（3条） 3. 潜在风险（2条） 4. 建议面试重点 5. 是否推荐进入下一轮

场景3：绩效面谈准备 - 背景：Q2绩效面谈季 - 角色：绩效教练 - 输入： - 员工绩效数据：[KPI完成率/360评估/项目成果] - 员工特点：[职级/入职时间/发展意愿] - 任务：生成绩效面谈指南 - 输出： 1. 面谈开场白 2. 亮点肯定（具体事例） 3. 改进建议（可执行） 4. 发展计划（3个月/6个月） 5. 资源支持需求 6. 常见抗性话术及应对

三、实操案例

案例1：批量简历筛选提效

背景： 校招季收到500+简历，3个HR手动筛选需1周，且标准不统一。

解决方案：

JD结构化Prompt

请将以下JD转化为筛选评分标准：
[粘贴JD]

要求：
1. 提取硬性条件（必须满足，一票否决）
2. 提取软性条件（加分项，分等级）
3. 制定评分规则（总分100分）
4. 标注需人工复核的模糊项

批量评估Prompt

请按以下标准评估候选人：
[粘贴评分标准]

候选人简历：
[粘贴]

输出：
- 总分：[ ]
- 硬性条件：[通过/不通过，原因]
- 核心优势：[ ]
- 风险点：[ ]
- 建议：[推荐/待定/淘汰]

效果： - 筛选时间：1周 → 2天 - 初筛准确率：70% → 90% - 面试官满意度：60% → 95%

案例2：薪酬竞争力分析

背景： 年底调薪，需评估公司薪酬竞争力，但缺乏市场数据。

解决方案：

市场数据收集Prompt

请帮我收集[城市+行业+岗位]的薪酬市场数据：
- 岗位：[列出5个核心岗位]
- 城市：[目标城市]
- 对标公司：[列出3-5家]

要求：
1. 分位值数据（25/50/75/90分位）
2. 薪酬结构（固薪/浮动/长期激励比例）
3. 福利对比
4. 近3年涨幅趋势
5. 标注数据来源及可靠性

内部公平性分析Prompt

基于以下内部薪酬数据：
[粘贴脱敏数据]

请分析：
1. 同岗同绩薪酬差异（合理范围 vs 异常值）
2. 新老员工倒挂情况
3. 部门间公平性
4. 调薪建议（重点人群/幅度/预算）
5. 风险预警（离职高风险人群）

效果： - 数据获取：无 → 有据可依 - 调薪决策：拍脑袋 → 数据驱动 - 员工满意度：65% → 85%

案例3：员工访谈辅助

背景： 离职率高，需做员工满意度访谈，但不知如何提问和分析。

解决方案：

访谈提纲生成Prompt

背景：公司[技术部门]离职率偏高，需做深度访谈
角色：组织发展顾问

请生成访谈提纲：
1. 开场建立信任（3个问题）
2. 工作体验（5个问题，含追问）
3. 管理评价（4个问题）
4. 发展诉求（3个问题）
5. 离职原因探测（3个问题，不直接问）
6. 建议收集（2个开放问题）

要求：
- 问题顺序符合心理安全建立逻辑
- 避免引导性提问
- 标注需重点关注的危险信号

访谈分析Prompt

以下是[5位]员工的访谈记录摘要：
[粘贴]

请分析：
1. 共性痛点（Top 3）
2. 部门差异
3. 紧急需处理事项
4. 中长期改进建议
5. 可快速见效的"速赢"措施

效果： - 访谈质量：随意聊天 → 结构化深度访谈 - 问题发现率：30% → 80% - 改进措施落地率：20% → 70%

五、常见误区与避坑指南

误区1：AI可以替代HR判断 → AI辅助筛选，但最终录用决策需人工面试确认。

误区2：用AI生成JD后直接发布 → 必须人工审核，避免歧视性描述和法律风险。

误区3：把员工敏感信息输入AI → 薪酬/绩效等敏感数据需脱敏处理，注意数据安全。

误区4：完全依赖AI的匹配评分 → 文化匹配、潜力等软性因素AI难以评估，需人工判断。
