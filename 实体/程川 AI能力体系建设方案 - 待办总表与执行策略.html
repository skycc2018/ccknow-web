程川 AI能力体系建设方案 - 待办总表与执行策略

基于《数字化素养提升项目》与《AI工具使用现状调研分析报告》整理

角色定位：AI能力体系 Owner（产品化负责人）

从项目定义看，你的角色其实不是"普通成员"，而是 AI能力体系 Owner（产品化负责人）： - 负责 工具标准制定 - 负责 课程体系设计 - 负责 工具选型决策 - 负责 AI能力产品化沉淀 - 支撑后续 组织级 AI 能力机制建设

待办总表（建议按优先级执行）

优先级 待办事项 当前问题（结合调研） 解决方案 输出物 完成时间
P0 校对访谈问题清单（支持张晓东） 访谈容易流于形式，无法挖真实业务痛点 增加"业务场景痛点 + AI替代机会 + 当前阻碍"三类追问模板 标准访谈SOP（15问） 立即
P0 确认公司统一AI工具栈 当前工具分散，缺统一认知；豆包/DeepSeek主导但高阶能力不足 制定三级工具矩阵：基础（全员）、专业（管理/HR）、进阶（产品/研发） 《公司AI工具白名单V1》 5/26
P0 确认国际模型使用策略（GPT / Claude） 员工明确诉求 Token / 稳定通道 给操昭阳明确采购方案（推荐 Team 统一采购） 账号采购方案 5/26
P0 输出《AI工具能力地图》 员工不知道"哪个工具干什么" 做工具能力二维图（能力 × 场景） 1页认知图 5/27
P1 梳理岗位化课程体系（3-5h） Prompt Engineering 是最大盲区（81%未系统掌握） 按岗位拆：管理、HR、运营、产品、数据 课程大纲PPT 5/27
P1 编写 Prompt 模板库 员工知道AI但不会提问 输出高频模板（汇报/分析/招聘/复盘） Prompt Handbook 5/28
P1 编写《减少幻觉操作指南》 幻觉认知弱，结果可信度低 建立"四步校验法"：上下文→约束→追问→验证 FAQ 手册 5/28
P1 支持操昭阳确定安装清单 工具过多，安装混乱 明确 必装 / 推荐 / 选装 分层 工具安装清单 5/28
P1 输出学习路径图 学习容易断层 入门→熟练→高手路径 学习路线图 5/29
P2 做一次公司级AI分享课 需要统一认知 用真实案例讲，不讲概念 培训课件 6/1
P2 沉淀"业务案例库" 缺组织复用机制 收集团队高光时刻做成 SOP AI案例库 6月
P2 设计 AI 能力认证机制 长期容易失效 分 Lv1-Lv4 认证 认证标准 6月
P3 识别首批智能体试点场景 已出现成熟案例（排产/薪酬/PRD） 选3个部门 PoC 智能体路线图 Q3

当前最关键的 4 个动作（必须先干）

1️⃣ 立即确定公司工具标准（最高优先）

当前工具分散，缺统一认知；豆包/DeepSeek主导但高阶能力不足

建议三级工具矩阵：

层级 目标人群 工具 解决场景
全员基础层 全体员工 豆包、DeepSeek 低门槛普及
专业增强层 管理/产品/HR ChatGPT Team、Claude Team 深度分析 / 长文推理 / 战略思考
创造层 产品/研发 Cursor、Figma AI AI Coding / 原型生成

2️⃣ 课程不要讲AI概念，要讲岗位打法

Prompt Engineering 是最大盲区（81%未系统掌握）

岗位化课程结构：

岗位 课程主题
管理层 用AI做经营决策
HR 招聘JD / 简历分析 / 绩效辅助
产品 PRD → 原型 → Demo
运营 活动方案 / 数据分析
销售 客户洞察 / 跟进话术

⚠️ 否则培训会失效。 不要讲概念，要讲每个岗位怎么用AI提效。

3️⃣ 重点解决 Prompt Engineering 空白

调研最危险的问题： - 33% 没听过 Prompt Engineering - 48% 听过但没系统学

这说明大家只是"会聊天"，不会"驾驭模型"。

建议输出：《菜洋洋 Prompt 标准模板》

统一六要素： 1. 背景 - 任务上下文 2. 角色 - AI扮演什么角色 3. 输入材料 - 提供什么信息 4. 输出格式 - 期望什么形式 5. 校验标准 - 质量要求 6. 迭代追问 - 如何优化

这会立刻拉开能力差距。

4️⃣ 不要做培训项目，要做能力操作系统

你真正应该推进的是：

工具标准化
    ↓
学习体系化
    ↓
模板资产化
    ↓
案例复用化
    ↓
智能体沉淀化
    ↓
能力认证化

这才是长期组织升级。

对程川个人价值（很重要）

如果这个项目做好，你在公司角色会从：

产品总监

升级成：

AI数字化能力建设负责人 / AI Transformation Owner

这是 2026 年非常稀缺的位置。

你不是在"搞培训"。

你是在帮公司建立：

组织级 AI Operating System（AIOS）
